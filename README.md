# AcadConnect

AcadConnect é uma rede social acadêmica desenvolvida como projeto da disciplina de Introdução à Engenharia.  
O sistema tem como objetivo conectar estudantes e pesquisadores através da publicação de artigos, comentários e interações acadêmicas.

---

# Objetivo do Projeto

O projeto foi criado com a proposta de oferecer um ambiente onde usuários possam:

- Publicar artigos acadêmicos
- Compartilhar conhecimento
- Interagir através de comentários
- Pesquisar conteúdos
- Entrar em contato com autores
- Criar conexões acadêmicas

---

# Funcionalidades

## Usuários
- Cadastro de usuários
- Login e logout
- Sessão de usuário
- Perfil acadêmico

## Artigos
- Publicação de artigos
- Visualização de artigos
- Edição de artigos
- Exclusão de artigos
- Feed dinâmico

## Interações
- Comentários em artigos
- Sistema de reações
- Compartilhamento de conteúdo
